# DevOps Projects

A collection of DevOps projects built with various related tools such as **Docker**, **Docker Compose**, **Terraform**, **Ansible**, **AWS EC2**, and **GitHub Actions**.

---

## Projects

| Project                              | Description                                                                                                                            | Repository                                                                   |
| ------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| **devops-todo-app**                  | A simple full-stack Todo application deployed with Docker, Terraform, and Ansible on AWS EC2.                                          | [View Repo](https://github.com/markfinerty/devops-todo-app)                  |
| **devops-dockerized-express**        | A Dockerized Node.js/Express API demonstrating container orchestration and CI/CD automation.                                           | [View Repo](https://github.com/markfinerty/devops-dockerized-express)        |
| **devops-qr-code**                   | A microservice for generating QR codes using Docker and GitHub Actions for automated builds and deployments.                           | [View Repo](https://github.com/markfinerty/devops-qr-code)                   |
| **devops-multi-service-application** | A multi-service application built with Docker, Docker Compose, Express.js, React.js, Nginx (reverse proxy), MongoDB and a Redis cache. | [View Repo](https://github.com/markfinerty/devops-multi-service-application) |

## Tools & Technologies

- **Docker** – Containerization of applications
- **Docker Compose** – Multi-container orchestration
- **Terraform** – Infrastructure as Code for AWS provisioning
- **Ansible** – Configuration management and server automation
- **AWS EC2** – Cloud compute hosting for deployed services
- **GitHub Actions** – Continuous Integration and Deployment workflows
